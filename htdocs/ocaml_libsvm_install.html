Installation

To build the ocaml-libsvm library you need to have:

  • ocaml-3.08
  • lacaml-3.06
  • findlib
  • libsvm-2.8

After compiling the libsvm library you need to do the following to complete the installation of the library:

  1. Create a shared library
    $ g++ -shared -o libsvm.so svm.o
  2. Install the library
    $ mkdir -p /usr/lib/libsvm
    $ cp libsvm.so /usr/lib/libsvm
  3. Install header file
    $ mkdir -p /usr/include/libsvm
    $ cp svm.h /usr/include/libsvm

Finally you can compile and install ocaml-libsvm with:
$ make && make install

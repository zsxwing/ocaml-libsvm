Introduction

OCaml-libsvm provides bindings of the libsvm library for the OCaml language.

Libsvm is an efficient implementation of Support Vector Machines for classification and regression tasks. Further information can be found at the official libsvm-homepage.

OCaml is a very popular variant of Standard ML. Further information can be found at the official OCaml-homepage.
